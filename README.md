# Learning Temporal

> Learning Temporal SDK

## Getting Started

### Install CLI

```shell
$ brew install temporal
```

### Run dev server

```shell
$ temporal server start-dev
```

### Access Web UI

```shell
$ open http://localhost:8233
```
